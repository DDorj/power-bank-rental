# Power Bank Rental Research

Power bank түрээсийн үйлчилгээ болон сүүлийн үеийн технологийн талаарх судалгааны материалын төсөл.

## Файлын бүтэц

- `RESEARCH.md` — Үндсэн судалгааны баримт (зах зээл, бизнес модель, технологи, эх сурвалжууд)
- `BACKEND.md` — Backend project эхлүүлэхэд шаардлагатай модуль, стек, интеграцуудын тодорхойлолт
- `PAYMENT.md` — Bonum QR pay ашиглан барьцаа hold загварлах payment architecture
- `INIT.md` — Project эхлүүлэх 10 phase бүхий ажлын дараалал
- `MOBILE_APP_STRUCTURE.md` — AI design tool-д зориулсан mobile app structure, screen map, station/power bank UX brief
- `AJLIIN_DAALGAVAR_60_HONOG.md` — Backend, mobile app, dashboard ажлын хүрээ болон 60 хоногийн timeline
- `.claude/CLAUDE.md` — Project-specific Claude Code заавар (rule)
- `.claude/agents/` — 5 тусгайлсан агент (bonum/dan/iot/nestjs/test)
- `.claude/hooks/` — Format, no-verify block, typecheck hook
- `.claude/settings.json` — Permissions + hook бүртгэл
- `README.md` — Төслийн тайлбар

## Хамрах хүрээ

1. Дэлхийн зах зээлийн хэмжээ ба өсөлтийн прогноз
2. Бизнес модель ба ROI тооцоо
3. Тоног төхөөрөмж ба программ хангамжийн архитектур
4. Батерейн (силикон-нүүрстөрөгч) ба GaN цэнэглэгчийн технологи
5. Тэргүүлэгч компаниудын харьцуулалт
6. Монголын зах зээлд нэвтрүүлэх боломж
